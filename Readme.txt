Prerequisites
This is the list of things you need to use the software.
   - PHP: >=7
   - MySQL
   - Composer

1. Import database "_wshop_api_shop.sql".
2. Install composer packages ("composer install")
3. Customize "database.ini" variables as needed to run the environment.
4. Run project ("php -S localhost:3000 -t public/")
5. Documentation is available at "https://documenter.getpostman.com/view/19818393/2s847EQDct"
